# Lumbago UI Kit — v1.0
Autor: Graphic Designer (AI)

Ten pakiet zawiera kompletny projekt UI dla aplikacji Lumbago Music AI:
- 9 ekranów hi-fi (SVG)
- 50 ikon (SVG, outline)
- pełny zestaw design tokens (dark + light)
- UI Kit (SCSS + opis komponentów)
- React Starter
- manifest, licencję, instrukcję budowy

## Instrukcja instalacji:
1. Zaimportuj folder `mockups/` do Figmy.
2. Zaimportuj ikony z folderu `icons/` do Figma → Components.
3. Utwórz Color Styles i Text Styles na podstawie `design-tokens/tokens-dark.json`.
4. Uruchom starter React:
   ```bash
   cd react-starter/
   npm install
   npm run dev
   ```
